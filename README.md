# Payroll App

## Introduction

This is the Payroll App, designed to manage and process payroll efficiently.

## Environment Variables

Ensure the following environment variables are set in your `.env` files:

- `SECRET_KEY=`
- `DEBUG=`
- `DB_NAME=`
- `DB_USER=`
- `DB_PASSWORD=`
- `DB_HOST=`
- `DB_PORT=`
